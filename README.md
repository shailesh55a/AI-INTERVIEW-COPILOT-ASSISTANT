# 🚀 AI Interview Copilot

AI-powered interview preparation platform built with Python, Streamlit, Groq LLM, SQLite, and Docker.

## Features

- Resume PDF Parsing
- Resume Analysis using AI
- Job Description Matching
- Interview Question Generation
- Answer Evaluation
- PDF Report Generation
- SQLite Database Storage
- Dockerized Deployment

## Tech Stack

- Python
- Streamlit
- Groq (Llama 3.3)
- SQLite
- ReportLab
- Docker
- GitHub

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
